# ⚛️ Certificación en React

React es la **librería más popular para construir interfaces web interactivas**. En este curso trabajamos con:

✅ **Componentes y Props**: Creación de interfaces modulares y reutilizables.  
✅ **Hooks**: Uso de `useState`, `useEffect` y otros hooks esenciales.  
✅ **Context API y Redux**: Gestión del estado global de la aplicación.  
✅ **Routing con React Router**: Creación de aplicaciones de múltiples páginas.  
✅ **Consumo de APIs y Firebase**: Integración con bases de datos en tiempo real.

El **proyecto final** consistió en un **e-commerce funcional**, donde implementamos carritos de compra y autenticación con Firebase. 🔥
