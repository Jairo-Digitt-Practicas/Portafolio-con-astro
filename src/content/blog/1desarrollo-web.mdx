# 🌐 Certificación en Desarrollo Web

Este curso es ideal para quienes inician en el mundo del desarrollo web. Durante el curso aprendimos:

✅ **HTML5 y Semántica**: Creación de estructuras de páginas accesibles.  
✅ **CSS3 y Flexbox**: Diseño moderno y responsivo.  
✅ **Bootstrap**: Uso del framework para crear interfaces atractivas.  
✅ **Git y GitHub**: Control de versiones y colaboración en proyectos.

Además, aplicamos estos conocimientos en un **proyecto final**, creando una página web completa desde cero. 🚀
